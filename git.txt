Comandos git aprendidos: 
- git init (inicializa uma pasta especifica como repositorio)
- git status (exibe o status do repositorio)
- git add "NomeDoArquivo" (adiciona determinado arquivo a lista de commits)
- git add . (adiciona TODOS arquivos a lista de commits)
- git commit -m "" (realiza o commit nos arquivos da lista)
- git push (adiciona a versão do commit a nuvem)
- git remote add origin https://github.com/Phdss04/Git  (defini um repositorio para upar os commits do git)
- git remote rm origin (comando para remover um repositorio)
- git reflog (permite visualizar as versões do codigo)
- git reset --hard id (permite voltar o codigo a uma versão anterior)
- git branch (mostra todas as branchs disponíveis)
- git branch "nome da branch" (permite criar uma nova branch)
- git checkout "nome da branch" (permite trocar de branch)
- git merge "nome da branch que deseja puxar as alterações"
obs: antes de usar o comando "git merge" utilizar o comando "git pull" para trazer todas as alterações do servidor para a máquina local.
